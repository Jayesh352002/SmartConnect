# SmartConnect
Smart Connect is Contact App using which user can save and manage contacts easily and smartly. User can access all contacts efficiently.

<br>
<b>Techologies Used</b>
<br><br>
Frontend - Angular <br>
Backend - SprigBoot<br>
Database - MySQL <br>
